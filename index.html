FAQ

Questions server owners ask

Does Relay ban people automatically?

No. Relay's default mode is review-queue only: it flags suspicious activity and waits for a moderator to approve or dismiss it before anything happens to a member.

Does Relay store my members' messages?

Relay is built to analyze moderation signals, not archive conversations. It does not intentionally store complete message contents. See the Privacy Policy for full detail.

What permissions does Relay need?

Only what the features you enable require — view channels and message history for detection, plus kick, timeout, ban, and manage-server permissions if you turn on the review-queue actions that use them.

Is Relay free?

Relay is currently free while in beta. If paid tiers are introduced later, pricing will be disclosed before anything is charged.
